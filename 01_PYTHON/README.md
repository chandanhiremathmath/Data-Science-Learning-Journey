# 🐍 Python for Data Science

This module covers the essential Python concepts required for Data Science, Machine Learning, and Artificial Intelligence.

## 📚 Topics Covered

- Variables and Data Types
- Operators
- Strings
- Lists
- Tuples
- Sets
- Dictionaries
- Conditional Statements
- Loops
- Functions
- Lambda Functions
- map(), filter(), reduce()
- File Handling
- Exception Handling
- Object-Oriented Programming (OOP)
- Practice Problems

## 📂 Files

| Notebook | Description |
|-----------|-------------|
| 01_Python_Basics.ipynb | Core Python concepts and syntax |
| 02_Functions_FileHandling.ipynb | Functions, Lambda, Modules, File Handling |
| 03_Exception_Handling.ipynb | Exception Handling with examples |
| 04_OOP_for_DataScience.ipynb | OOP concepts relevant to Data Science |
| Practice_Problems.ipynb | Interview and coding practice questions |

## 🎯 Learning Outcome

After completing this module, you will be able to:

- Write clean Python programs.
- Work confidently with Python data structures.
- Build reusable functions.
- Handle files and exceptions.
- Understand OOP fundamentals.
- Prepare for Data Science coding interviews.
